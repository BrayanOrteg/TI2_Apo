# TI2_Apo
Para insertar un país o una ciudad el nombre puede ser máximo de 2 palabras.
Si ocurre un error en la lectura del SQL dejara de leer el resto de entradas.
La población de la ciudad debe ir escrita como int ejemplo: 2,  y la población de los paises como double ejemplo: 2.2
Las variables que pueden ser filtradas y ordenadas para las ciudades son: id, name, population, country.
Las variables que pueden ser filtradas y ordenadas para los paises son: id, name, population, code.
